# doom

My Doom Emacs configuration.

## Usage

```
# first of all, install doom emacs
git clone --depth 1 https://github.com/doomemacs/doomemacs ~/.config/emacs
~/.config/emacs/bin/doom install

# then, clone my own repo
rm -rf ~/.config/doom # remove default config
git clone https://github.com/UmutKzl/doom.git ~/.config/doom
doom sync
```
